# 🚀 React To-Do App

A modern and responsive To-Do List web application built using React.  
It helps users efficiently manage daily tasks with a clean and interactive interface.

---

## 🌐 Live Demo

🔗 https://yourdailyitask.netlify.app/

---

## ✨ Features

- ➕ Add tasks  
- ❌ Delete tasks  
- ✅ Mark tasks as completed  
- 📱 Fully responsive design  
- ⚡ Smooth and fast user experience  

---

## 🛠️ Tech Stack

- React.js  
- JavaScript (ES6+)  
- HTML5 & CSS3
